# 20240229_Rese

＃アプリケーション名
Rese（リーズ）

＃＃作成した目的
Webサービス制作の概要・方針決定
サービス概要		ある企業のグループ会社の飲食店予約サービス
制作の背景と目的	外部の飲食店予約サービスは手数料を取られるので自社で予約サービスを持ちたい。
制作の目標		初年度でのユーザー数10,000人達成
作業範囲		設計・コーディング・テスト
納品方法		GitHubでのコードの共有とサーバー設置
